# Spotlight AI Popup for Local LLMs

A lightweight, global Spotlight-style macOS overlay that connects to local LLMs via LM Studio. Press a hotkey from anywhere on your Mac to bring up the UI, ask a coding question, and get streamed responses instantly.

## Prerequisites
* **macOS** (Built specifically with Mac window attributes)
* **LM Studio** running a local server on `http://localhost:1234`
* Python 3.9+
